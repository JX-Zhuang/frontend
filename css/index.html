盒模型

  • 块级盒子
    • 占据父容器的可用空间
    • 盒子会换行
    • width、height都生效
    • padding、margin、border会把其他的元素从当前盒子周围“推开”
  • 内联盒子
    • 盒子不会换行
    • width、height不生效
    • 垂直方向的padding、margin、border会被应用，但是不会把其他的元素从当前盒子周围“推开”
    • 水平方向的padding、margin、border会被应用，会把其他的元素从当前盒子周围“推开”

层叠与继承

  • 千位：style
  • 百位：id
  • 十位：类选择器、属性选择器、伪类
  • 个位：元素、伪元素
  • 千、百、十、个，不会进位
  • !important 优先级最高

布局和包含块

  • 确定包含块
    • 如果position是static、relative、sticky，最近的祖先块级元素的内容区的边缘组成
    • 如果position是absolute，包含块是由它的最近的position的值不是static的祖先的内边距的边缘组成
    • 如果position是fixed，包含块是viewport
    • 如果position是absolute或fixed，包含块可能是满足以下条件的最近父元素的内边距的边缘组成
      • transform或perspective不是none
      • will-change的值是transform或perspective
      • filter不是none，或will-change是filter
      • contain是paint
  • 根据包含块计算百分值
    • height、top、bottom，是通过包含块的height的值。如果包含块的height，会根据内容变化，而包含块的position是relative或static，那么这些值是auto。
    • width、left、right、margin、padding是根据包含块的width来计算的。

CSS 的值与单位

  • 绝对单位
  • 相对单位
    • em：font-size相对于父元素的字体大小，其他属性相对于自身字体大小
    • rem：相对于根元素的字体大小
    • vw：视窗宽度的1%
    • vh：视窗高度的1%

块格式化上下文(BFC)

  • 根元素
  • 浮动元素
  • position是absolute、fixed
  • overflow不是visible、clip的块元素
  • 包含内部浮动
  • 排除外部浮动
  • 阻止外边距重叠

flex

  • 主轴
  • 交叉轴
  • flex
    • flex:number
      • flex-grow:number
      • flex-shrink:1
      • flex-basis:0%
    • flex:none
      • flex-grow:0
      • flex-shrink:0
      • flex-basis:none
    • flex:auto
      • flex-grow:1
      • flex-shrink:1
      • flex-basis:auto
  • flex-grow：增长系数
  • flex-shrink：收缩系数
  • flex-basis：元素在主轴方向上的初始大小
  • 对齐属性
    • justify-content：如何分配顺着弹性容器主轴的元素之间及其周围的空间
    • align-items：交叉轴方向上的对齐方向
    • align-self：会对齐当前grid或flex行中的元素，覆盖align-items的值
    • align-content：如何沿着弹性盒子布局的纵轴和网格布局的主轴在内容项之间和周围分配空间。对但行弹性盒子模型无效
